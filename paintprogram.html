Paint and Photo Manipulation Program

I helped develop a paint and photo editing application as a part of my senior project design course at the University of Minnesota. The program allows users to apply drawing tools, images, filters, and other effects to a digital canvas. Additionally, users can save or load previous canvas states or use undo and redo features to modify the canvas. As an example, mouseover the images see to see two filters applied to a photo of the Minneapolis skyline.

Visit my Github page for a look at the code for this project.
